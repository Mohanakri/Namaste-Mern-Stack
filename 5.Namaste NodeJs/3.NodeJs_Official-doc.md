## üß† What Node.js Actually Is

* **Node.js** is *not a framework* or *library*.
* It‚Äôs a **runtime environment** that lets you run **JavaScript outside a browser**, on your computer or server.
* It uses **Chrome‚Äôs V8 engine** (the same JS engine Chrome uses) but adds:

  * File system access
  * Networking
  * OS interaction
  * Package management (via npm)

So while React or Express run *on top of Node.js*, Node.js itself is the foundation.
