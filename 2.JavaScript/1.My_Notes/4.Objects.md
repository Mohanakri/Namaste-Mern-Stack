# 🧠 JavaScript Objects — All-in-One Notes

---

## 🔹 1. Object Basics

| Concept                         | Description                                                        |
| ------------------------------- | ------------------------------------------------------------------ |
| **Object Definition**           | Collection of key-value pairs representing properties and methods. |
| **Object Literal**              | Simplest way to create an object using `{}` syntax.                |
| **Accessing Properties**        | Use dot `.` or bracket `[]` notation.                              |
| **Adding/Modifying Properties** | Add or update properties dynamically.                              |
| **Deleting Properties**         | Use `delete` keyword to remove a property.                         |

```js
const person = { name: "Mohana", age: 25 };
console.log(person.name);
person.city = "Chennai";
delete person.age;
```

---

## 🔹 2. Object Creation Methods

| Concept                  | Description                                            |
| ------------------------ | ------------------------------------------------------ |
| **Object Constructor**   | Create an object using `new Object()`.                 |
| **Object.create()**      | Creates an object with a specific prototype.           |
| **Factory Function**     | Function returning new objects.                        |
| **Constructor Function** | Function used with `new` to create multiple instances. |

```js
const obj1 = new Object();
const proto = { greet() { console.log("Hi"); } };
const obj2 = Object.create(proto);
function makeUser(name) { return { name }; }
function User(name) { this.name = name; }
```

---

## 🔹 3. Object Properties

| Concept                     | Description                                              |
| --------------------------- | -------------------------------------------------------- |
| **Property Shorthand**      | When variable name and property name are same.           |
| **Computed Property Names** | Dynamic property keys using `[]`.                        |
| **Property Descriptors**    | Control property behavior using `Object.defineProperty`. |

```js
const name = "Mohana";
const user = { name }; // shorthand
const key = "age";
user[key] = 25;
Object.defineProperty(user, "city", { value: "Hyderabad", writable: false });
```

---

## 🔹 4. Object Methods

| Concept                        | Description                                       |
| ------------------------------ | ------------------------------------------------- |
| **Methods in Objects**         | Functions inside objects.                         |
| **`this` Keyword**             | Refers to the current object context.             |
| **Arrow Functions in Objects** | Don’t have their own `this`.                      |
| **Method Shorthand**           | Shorter syntax for defining functions in objects. |

```js
const car = {
  brand: "Tesla",
  start() { console.log(this.brand + " started"); },
  stop: () => console.log("Stopped") // no this binding
};
car.start();
```

---

## 🔹 5. Object Iteration

| Concept              | Description                          |
| -------------------- | ------------------------------------ |
| **for...in Loop**    | Iterates over enumerable properties. |
| **Object.keys()**    | Returns array of property names.     |
| **Object.values()**  | Returns array of property values.    |
| **Object.entries()** | Returns array of key-value pairs.    |

```js
const obj = { a: 1, b: 2 };
for (let key in obj) console.log(key);
console.log(Object.entries(obj));
```

## 🔹 6. Object Cloning & Merging

| Concept                            | Description                                            |
| ---------------------------------- | ------------------------------------------------------ |
| **Shallow Copy (`Object.assign`)** | Copies properties from one or more objects.            |
| **Spread Syntax (`...`)**          | Another way to clone/merge objects.                    |
| **Deep Copy**                      | Copies nested objects (needs structuredClone or JSON). |

```js
const a = { x: 1 }, b = { y: 2 };
const c = Object.assign({}, a, b);
const d = { ...a, ...b };
const deep = structuredClone({ nested: { n: 5 } });
```

## 🔹 7. Object Comparison

| Concept                  | Description                                      |
| ------------------------ | ------------------------------------------------ |
| **Reference Comparison** | Objects are compared by reference, not by value. |
| **Shallow Equality**     | Compare keys and values manually.                |

```js
const obj1 = { a: 1 };
const obj2 = { a: 1 };
console.log(obj1 === obj2); // false
```

## 🔹 8. Object Destructuring

| Concept                   | Description                             |
| ------------------------- | --------------------------------------- |
| **Basic Destructuring**   | Extract properties into variables.      |
| **Default Values**        | Assign defaults for missing properties. |
| **Nested Destructuring**  | Extract nested object values.           |
| **Rest in Destructuring** | Collect remaining properties.           |

```js
const user = { name: "Mohana", age: 25, city: "Pune" };
const { name, age, city = "NA" } = user;
const { ...rest } = user;
```

## 🔹 9. Object Prototypes & Inheritance

| Concept                                                   | Description                                         |
| --------------------------------------------------------- | --------------------------------------------------- |
| **Prototype**                                             | Shared object from which others inherit properties. |
| **`Object.getPrototypeOf()` / `Object.setPrototypeOf()`** | Access or modify prototype chain.                   |
| **Prototype Chain**                                       | Lookup chain for methods and properties.            |
| **Class Syntax (ES6)**                                    | Modern way to handle inheritance.                   |

```js
const parent = { greet() { console.log("Hello"); } };
const child = Object.create(parent);
child.greet();

class Animal { speak() { console.log("Sound"); } }
class Dog extends Animal { speak() { console.log("Bark"); } }
```

---

## 🔹 10. Object Utilities

| Concept                                | Description                                                  |
| -------------------------------------- | ------------------------------------------------------------ |
| **Object.freeze()**                    | Makes an object immutable.                                   |
| **Object.seal()**                      | Prevents adding/removing properties but allows modification. |
| **Object.isFrozen() / isSealed()**     | Checks freeze or seal status.                                |
| **Object.hasOwn() / hasOwnProperty()** | Checks if property exists on object itself.                  |

```js
const user = { name: "Mohana" };
Object.freeze(user);
console.log(Object.isFrozen(user));
```

## 🔹 11. JSON & Objects

| Concept              | Description                          |
| -------------------- | ------------------------------------ |
| **JSON.stringify()** | Converts object to JSON string.      |
| **JSON.parse()**     | Converts JSON string back to object. |

```js
const obj = { x: 1 };
const str = JSON.stringify(obj);
const newObj = JSON.parse(str);
```
